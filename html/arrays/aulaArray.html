Aula de Arrays em JavaScript

1. Introdução aos Arrays

O que é um Array? Arrays são listas ou coleções de dados em JavaScript. Eles permitem armazenar múltiplos valores em uma única variável.

let frutas = ["maçã", "banana", "laranja"];

2. Acessando Elementos

Em JavaScript, os Arrays são indexados a partir de 0. O primeiro item tem índice 0, o segundo índice 1, e assim por diante.

console.log(frutas[0]); // Saída: "maçã"

3. Principais Métodos de Arrays

  • push() - Adiciona um elemento ao final do Array.
  • pop() - Remove o último elemento do Array.
  • unshift() - Adiciona um elemento ao início do Array.
  • shift() - Remove o primeiro elemento do Array.
frutas.push("uva"); // Adiciona "uva" ao final
frutas.pop(); // Remove o último elemento

4. Iterando em Arrays

Você pode usar um loop for básico para percorrer um Array, ou métodos como forEach() e map():

for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}

// Usando forEach
frutas.forEach((fruta) => console.log(fruta));

5. Exemplos Práticos

Mostre como o filter(), map() e reduce() podem ser usados para manipular dados em Arrays.

let numeros = [1, 2, 3, 4, 5];
let pares = numeros.filter(num => num % 2 === 0); // [2, 4]

6. Dicas e Boas Práticas

Evite Arrays aninhados complexos: Arrays dentro de Arrays podem ser confusos para iniciantes.
