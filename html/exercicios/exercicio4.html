Exercícios Básicos de Arrays em JavaScript

1. Criação e Acesso aos Arrays

Crie um Array com pelo menos 5 nomes de frutas e acesse o terceiro elemento. Em seguida, altere o último elemento do Array para uma fruta diferente.

let frutas = ["maçã", "banana", "laranja", "uva", "pêssego"];
console.log(frutas[2]); // Saída: "laranja"
frutas[frutas.length - 1] = "morango";
console.log(frutas);

2. Manipulação Simples de Arrays

Adicione uma nova fruta ao final do Array usando o método push(). Remova o primeiro elemento usando shift() e adicione uma nova fruta ao início usando unshift().

frutas.push("melancia");
frutas.shift();
frutas.unshift("abacaxi");
console.log(frutas);

3. Iteração

Crie um Array com 5 números e use um loop for para exibir cada número no console. Em seguida, usando forEach(), exiba os elementos do Array de frutas em uma frase como "Eu gosto de [fruta]".

let numeros = [1, 2, 3, 4, 5];
for (let i = 0; i < numeros.length; i++) {
    console.log(numeros[i]);
}

frutas.forEach((fruta) => console.log("Eu gosto de " + fruta));

4. Métodos Simples de Arrays

Crie um Array de cores e inverta a ordem usando reverse(). Em seguida, use slice() para criar uma cópia do Array contendo apenas os primeiros 3 elementos.

let cores = ["vermelho", "azul", "verde", "amarelo", "roxo"];
cores.reverse();
console.log(cores);

let primeirasCores = cores.slice(0, 3);
console.log(primeirasCores);
